🌤️ Weather App

A simple React.js Weather Application built with Vite, Bootstrap, and Material UI.
It fetches real-time weather data from the Open-Meteo API (no API key required) and displays the temperature, wind speed, and weather conditions for any searched city.

✨ Features

🌍 Search weather by city name

🌡️ Displays current temperature, wind speed, and weather code

🎨 Styled using Bootstrap & Material UI

⚡ Built with Vite for fast development

🚀 Setup Instructions
1. Clone the Repository : 
cd weather-app

2. Install Dependencies :
npm install

3. Run Development Server :
npm run dev


Then open 👉 http://localhost:5173/ in your browser.

🛠️ Tech Stack

React.js (UI library)

Vite (build tool)

Bootstrap 5 (styling)

Material UI (MUI) (UI components)

Open-Meteo API (weather data)

🌍 Example Usage

Enter any city name (e.g., London, Hyderabad, New York).

Click Search.

See real-time weather details displayed in a card.
